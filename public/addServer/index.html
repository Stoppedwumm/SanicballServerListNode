How to deploy an server

Step 1: Choosing your executable

Windows Download: Here

Linux Download: Here

For Macs: Download the Windows Executable

Step 2: Setup NGROK

Create your free ngrok account here: https://dashboard.ngrok.com/signup

Setup the agent according to the website (https://dashboard.ngrok.com/get-started/setup/) (you need an terminal)

Skip the ngrok http http://localhost:8080 step as we don't need it

Run the following command in your terminal:

ngrok tcp 25000

You should see something like this

You can see the public IP and port, where

IP: 6.tcp.ngrok.io

Port: 16142

Do not close the terminal window, it's the tunnel between your server and the internet

Step 3: Running the Server

Extract the zip file into any folder

Tutorial

Windows: Execute the SanicballServer.exe file.

Linux: Execute the SanicballServer file.

Mac

Download Homebrew

Open Terminal and type: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Install wine via brew install --cask wine-stable

Run the server via wine SanicballServer.exe

Step 4: Configuring the server

When you start the server, you will be asked some questions

Enter the server name and press ⏎ (Enter/Return)

Just press ⏎ (Enter/Return) to use the default port

Enter the max player count (1-64) and press ⏎ (Enter/Return)

Enter if you want to enable server lists (type yes or no) and press ⏎ (Enter/Return) (people can still connect to your server without a server list via the ngrok details)

If you chose to enable server lists, enter https://sanicball-server-list.vercel.app and press ⏎ (Enter/Return)

Enter the IP from NGROK (NOT THE PORT) and press ⏎ (Enter/Return)

Enter the port from NGROK and press ⏎ (Enter/Return)

My example config

Step 5: Done

Congratulations, your server is now running and added to the server list!

Check if your server is listed here: https://sanicball-server-list.vercel.app/list

To stop the server, just close the terminal window
